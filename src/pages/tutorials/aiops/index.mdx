# Cloud Pak for Watson AIOps Content

**[Intoduction](/tutorials/aiops/introduction)**: Introductory material that includes concepts, major features and an introduction to the lab environment and demo app.

**[Installation](/tutorials/aiops/installation)**: Installation guidence for AI Manager, Event Manager (on-line, air-gapped and hybrid), and AWX (the open source verion of Ansible Tower).

**[Connections](/tutorials/aiops/connections)**: Discusses configuration options for AIOps components. Provide guidence on configuring Slack ChatOps, Log Connectors, Instana Connector, Topology Observers, and ServiceNow connector.

**[Resource Management](/tutorials/aiops/resources/resource-management)**: Discusses the concepts of topological resources and how they map to incoming data (metrics, alerts and logs).  Instana connector defined resources are explained.  Topology rules are also discussed.

**[Policies](/tutorials/aiops/policies/introduction)**: Covers policies for associating runbooks to alerts and when to create stories.

**[Runbook Automation](/tutorials/aiops/automations/ssh)**: Discusses runbooks and automations (SSH, HTTP and Ansible Tower) for both AI Manager and Event Manager.

**[Event Manager Configuration](/tutorials/aiops/eventmanager-configuration/onprem-probe)**: Discusses Evetn Manager configuration options (probes, web hooks, and topologies).

**[Hints Tips](/tutorials/aiops/hintstips/aiops-architecture)**: Provide architecture guidance on how to set up AIOps. Other topics include using EFK stack, offline training, pre-trained models, and tips for debugging the cloud pak.  Much of this content originated from the [AIOps Up and Running](https://pages.github.ibm.com/up-and-running/watson-aiops/) site.
