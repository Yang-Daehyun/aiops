## AIOps Track

<PageDescription>

This site contains lab tutorials and suplemental materials to be used in the Watson AIOps Tech Jam 2022, which is sponsored by the Automation SWAT team.  Materials included cover Watson AIOps Event Manager, Watson AIOps AI Manager, Instana, and Turbonomic.
The tutorials provide a step-by-step set of instructions and are supplemented with additional reference materials.

The content is organized into 4 sections that cover the majority of the Watson AIOps family of products \(Cloud Pak for Watson AIOPs, Instana and Turbonomic\).  Each section includes sections on installation and configuration as well as instructions on how to use the product for key scenarios.  The sections MUST be executed in the order documented.  However, the Instana and Turbonomic sections do not have any dependencies on the Watson AIOps sections of the lab.

</PageDescription>
